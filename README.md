# Prank Page - 521 整蛊网页

一个看起来温馨的 521 友谊表白网页，实际暗藏 jump scare 视频整蛊。

## 使用方法

1. 把你的 jump scare 视频文件改名为 `scare.mp4`，放到这个文件夹里
2. 把这个文件夹 push 到 GitHub 仓库
3. 在仓库 Settings → Pages 中启用 GitHub Pages（选择 main 分支）
4. 把生成的链接发给你的朋友 😈

## 文件结构

```
prank-page/
├── index.html    # 主页面
├── scare.mp4     # 你的 jump scare 视频（需要自己添加）
└── README.md
```
